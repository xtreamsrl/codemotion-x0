# Create your own v0 exploiting generative AI

## Setup
- Install backend project dependencies
```shell
cd backend
pnpm i
```
- Install frontend project dependencies
```shell
cd frontend
pnpm i
```
- Copy `sample.env` in `config.env`
- Paste the OpenAI key from the chat into `config.env`
